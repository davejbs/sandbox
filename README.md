# Sandbox

## What is this project?

This is a kind of sandbox containing sub-projects. Well, I say sub-projects, but they're more like lightweight, reusable components and features. As for the name "sandbox", it works for now, I'll see if I come up with something catchier.

## Why this project?

Simply put, I sometimes get ideas and the urge to build features and components, but quickly lose motivation at the thought of having to set up a whole project just to test them. So rather than having messy code scattered across dozens of repositories, I decided to bring everything together here in an organized way.

Sometimes when I want to try things out, instead of building them in a vacuum, I leave them here, ready to be reused in other projects.

## What does it contain?

Features and widgets I've always wanted to build myself to sharpen my skills.

## Is it original?

Honestly, it depends on the feature. Some of them already exist out there.

## Is it perfect?

No, I'm still a student learning the ropes. I experiment, and if I pick up new patterns or conventions, I'll come back and update what needs to be updated.

## Can you use it?

Feel free! I just ask to be credited for the more substantial features (this will be noted on the page linked to each feature).
